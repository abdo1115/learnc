Introduction à C#

C# (prononcé "C sharp") est un langage de programmation moderne, orienté objet, développé par Microsoft dans le cadre de sa plateforme .NET. Il est utilisé pour créer :

  • Applications de bureau
  • Applications web
  • Applications mobiles
  • Jeux vidéo (avec Unity)
  • Applications d'entreprise

C# combine la puissance de C++ avec la simplicité de Visual Basic, tout en offrant des fonctionnalités modernes comme la gestion automatique de la mémoire.

Installation de l'environnement

Pour commencer à programmer en C#, vous avez besoin de :

  1. Visual Studio ou Visual Studio Code

    Téléchargez Visual Studio Community (gratuit) sur le site officiel de Microsoft.

  2. .NET SDK

    Installez le SDK .NET pour compiler et exécuter vos programmes C#.

Visual Studio inclut tout ce dont vous avez besoin pour commencer à développer en C#.

Les bases de C#

Voici un premier programme C# classique :

using System;

namespace MonPremierProgramme
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Bonjour, monde !");
            Console.ReadLine();
        }
    }
}

Explications :

  • using System; - Importe les fonctionnalités du namespace System
  • namespace - Organise votre code
  • class Program - Définit une classe nommée Program
  • static void Main() - Point d'entrée de votre programme
  • Console.WriteLine() - Affiche du texte dans la console
  • Console.ReadLine() - Attend une entrée de l'utilisateur

Variables et types de données

C# est un langage fortement typé. Voici les types de base :

  • int - Nombres entiers (ex: 42)
  • double - Nombres à virgule flottante (ex: 3.14)
  • bool - Valeurs booléennes (true/false)
  • char - Caractères uniques (ex: 'A')
  • string - Chaînes de caractères (ex: "Bonjour")

Exemple d'utilisation des variables :

int age = 25;
double taille = 1.75;
bool estEtudiant = true;
char grade = 'A';
string nom = "Jean Dupont";

Console.WriteLine($"Nom: {nom}, Age: {age}, Taille: {taille}m, Étudiant: {estEtudiant}, Grade: {grade}");

Structures conditionnelles

Les structures conditionnelles permettent d'exécuter du code selon certaines conditions.

If-else

int age = 18;

if (age >= 18)
{
    Console.WriteLine("Vous êtes majeur");
}
else
{
    Console.WriteLine("Vous êtes mineur");
}

Switch

char grade = 'B';

switch (grade)
{
    case 'A':
        Console.WriteLine("Excellent");
        break;
    case 'B':
        Console.WriteLine("Bien");
        break;
    case 'C':
        Console.WriteLine("Passable");
        break;
    default:
        Console.WriteLine("À améliorer");
        break;
}

Boucles

Les boucles permettent de répéter des actions.

Boucle for

for (int i = 0; i < 5; i++)
{
    Console.WriteLine($"Itération {i}");
}

Boucle while

int compteur = 0;
while (compteur < 5)
{
    Console.WriteLine($"Compteur: {compteur}");
    compteur++;
}

Boucle foreach

string[] fruits = { "pomme", "banane", "orange", "kiwi" };

foreach (string fruit in fruits)
{
    Console.WriteLine(fruit);
}

Méthodes (Fonctions)

Les méthodes sont des blocs de code réutilisables :

static void AfficherMessage(string message)
{
    Console.WriteLine(message);
}

static int Additionner(int a, int b)
{
    return a + b;
}

// Appel des méthodes
AfficherMessage("Bonjour !");
int resultat = Additionner(5, 3);
Console.WriteLine($"5 + 3 = {resultat}");

Classes et Objets

C# est un langage orienté objet qui utilise des classes pour créer des objets :

// Définition d'une classe
class Personne
{
    // Propriétés
    public string Nom { get; set; }
    public int Age { get; set; }
    
    // Constructeur
    public Personne(string nom, int age)
    {
        Nom = nom;
        Age = age;
    }
    
    // Méthode
    public void SePresenter()
    {
        Console.WriteLine($"Bonjour, je m'appelle {Nom} et j'ai {Age} ans.");
    }
}

// Création et utilisation d'objets
Personne personne1 = new Personne("Marie", 30);
Personne personne2 = new Personne("Pierre", 25);

personne1.SePresenter();
personne2.SePresenter();
